3.3.1.4.1.2 Drawable Info Box for Real Drawable

Figure 3.14: Drawable Info Box for real state and arrow. The Drawable Info Box for the arrow shows the messsage size, 1600 byte, and tag ID, 454.

Similarly for real drawables, Drawable Info Box can be brought up by right mouse clicking on the real drawables. In Figure 3.14, Drawable Info Boxes for a real arrow and a real state are shown. The Drawable Info Box for the arrow is invoked by clicking anywhere within the vicinity of the arrow body3.4, and the info box shows the starttime, start timeline ID, endtime, and ending timeline ID and some extra information implemented by the native format . In this example, the message size carried by the specific arrow is 1600 byte.
