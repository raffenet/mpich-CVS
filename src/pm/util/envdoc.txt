This file contains documentation for the various features supported by the 
PMI utility routines.  These are specified as doctext "Name" blocks, so that
they may be easily included into documentation.

/*N PORTRANGE

. MPIEXEC_PORTRANGE - Specifies the allowed range of ports.  The value is
.vb
      low [: high ]
.ve
 where the valid ports lie in the range 'low' to 'high' inclusive.  If only 
 one value is provide, the is the only port available.

N*/
